# NextJS

A React Framework that supports:

- SSR
- Routing
- Code splitting
- API Routes
- and [more](https://nextjs.org/#more)!

---

# Now

A Zero config cloud deployment tool supports:

- Auto deployment
- Free auto SSL
- Serverless Functions
- Static Hosting
- Edge caching
- and [more](https://now.sh)!

---

# Why

> It is the fastest way to prototype a web app,
> and get it to a stable, scalable production environment.
